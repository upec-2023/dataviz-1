A l'aide de la fonction histogram du module numpy, nous pouvons crer les listes d'abscisses et d'ordonnes pour instancier l'histogramme de l'chantillon.

Cette fonction a 3 paramtres:

  • a : Prend en argument une liste ou un array.
  • density : Prend en argument un boolen. Si True, la fonction va gnrer une liste de frquences. Sinon, la fonction va gnrer une liste d'occurences.
  • bins: Le nombre de barres de l'histogramme.

Elle renvoie 2 listes:

  • hist: array des valeurs en ordonne de l'histogramme.
  • bin_edges: array des coordonnes en abscisse des bords des barres de l'histogramme.

Grce ces listes, nous pouvons instancier une source pour notre histogramme.

  • Lancer la cellule suivante pour gnrer les listes d'abscisses et d'ordonnes et la source pour l'histogramme.
In[5]:
from bokeh.models import ColumnDataSource

# Listes d'abscisses et d'ordonnes

hist, bin_edges = np.histogram(a = x,             # chantillon
                                density = True,   # pour obtenir un array de frquences et non d'occurences
                                bins = 50)        # nombre de barres de l'histogramme

source = ColumnDataSource({
        'hist' : hist,
        'x' : bin_edges[:-1]      # l'array bin_edges contient un lment de plus que hist. Il faut enlever le dernier pour 
})                                # avoir deux array de mme taille

Cration d'une bande

La classe Band est un type d'annotation qui permet de surligner une bande autour de donnes choisies. Une utilisation courante pour l'annotation Band est de surligner la zone d'incertitude lie une srie de mesures.

Nous allons commencer par crer une srie temporelle de points alatoires.

  • Importer les modules pandas et numpy.

  • Crer les listes alatoires avec suivantes :

y = np.random.normal(size = 2500) * 2 + 5

Pour cela, il suffira d'utiliser une fentre coulissante de longueur 100 grce aux mthodes df1["y"].rolling(window = 100).agg({"y_mean": np.mean, "y_std": np.std}.

Les 100 premires entres de df2 seront des NA puisqu'il n'est pas possible de calculer la moyenne ou l'cart-type sur les 100 observations prcdentes.
